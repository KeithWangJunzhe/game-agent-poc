# Match Reports

This directory stores JSON match reports generated by the runner.

Each report includes:

- initial state
- command log
- command results
- final state
- after-match stats
- raw `trajectory` JSON
- agent decision counters such as `action_counts` and `source_counts`

Reports are generated automatically when a run finishes.
